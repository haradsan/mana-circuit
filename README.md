# ✦ マナサーキット ✦

カードゲーム × ボードゲームの対戦ゲーム（vs CPU／🎮2人対戦）。
セガサターンの名作『カルドセプト』へのオマージュ作品です。

## 遊び方

**`index.html` をブラウザで開くだけ**（インストール・通信不要、オフラインで動きます）。

### ルール概要
- 🎲 ダイスを振って周回ボードを移動
- 空き地に止まったら手札のクリーチャーを召喚して土地を確保
- 敵の土地では「通行料を支払う」か「クリーチャーで侵略バトル」
- 同じ属性（🔥火・🌳木・⛰️地・💧水）の土地を集めると**連鎖**で通行料が倍増
- 自分の土地はレベルアップで価値と通行料アップ。**クリーチャーの交代**も可能
- 隣接する自軍領地で防衛クリーチャーが強くなる（**土地の援護** ST+10〜最大+40）
- ⛩️関門を規定数（多くのステージで3つ）そろえて🏰城を**通過または停止**すると周回ボーナス＝魔力＋自軍全回復（劣勢時は魔力1.5倍の「逆転の風」）
- 🏰城に**ぴったり停止（またはリコール）**すると、周回に関係なく**領地コントロール**＝全自領を対象に1回行動できる
- 1ターンの能動行動は1回。停止マスで行動しなかったターンは、通過した自領について
  **🏇侵攻・🔄交代・⬆レベルアップ**のどれか1つを行える（②通過アクション。出発マスは対象外）。
  自分の土地に到達したときは、レベルアップ・交代に加えて駐留クリーチャーの**🏇侵攻**も選べる
- 選択中でないときに盤面のマスをクリックすると、**駐留クリーチャーの能力・実効防衛値・通行料**などの情報を確認できる
- **ステージごとの目標資産に達して城に帰還したら勝利！**（相手を破産させても勝ち）
- 自分/相手のステータス・手札は画面に**固定ウィンドウ**で常時表示（各✕で一時的に隠せる）。
  自分の番でダイスを振る前は🏳投了でゲームを中断できる

### ステージ（全11面）
クリアすると次のステージが解放される（進行はブラウザに自動保存）。
**盤面の形とマス数（16〜37）が全ステージで異なる**: 小さな環、細長い高地、
橋や尾根の近道（分岐）、2つの環が城で交差する「八の字」、縦横の谷道が交わる「十字路」、
星型の「四つ辻」、魔王城への地獄回廊など。ワープ🌀・マグマ🌋・疾風💨の特殊マスや、通行料レート・
CPUの性格・得意属性もステージごとに変化する。分かれ道ではルートを見て進む方向を選べる。

### バトル
- **属性相性**: 🔥火 → 🌳木 → ⛰️地 → 💧水 → 🔥火（左が右に強い、ST+10。＝水＞火＞木＞地＞水）
- 低確率で**💫会心の一撃**（ダメージ1.5倍）が発生。格上にも一発逆転の目がある
- **戦闘後は残りHPのまま駐留**し傷が残る（周回で全回復）
- 特殊能力: **先制**／**貫通**／**強襲**（侵略時ST+20）／**守護**（防衛時HP+20）／**豪運**（会心率25%）／
  **捕縛**（撃退した侵略者を1ターン拘束）／**不動**（侵略・侵攻に出せない防御専用・高HP）／
  **護法**（敵の対象指定スペルの対象にならない）／**連撃**（バトルで続けて2回攻撃）／
  **物理無効**（物理攻撃が効かない）／**物理反射**（物理攻撃をそっくり跳ね返す）／**魔法攻撃**（攻撃が魔法＝無効・反射を貫く）／
  **模倣**（バトル時、相手の基本ST・HP・能力をそっくり写し取って戦う・v17）

### カード（全123種・レア度つき）
- クリーチャー78種（火17/木17/地17/水19/**⚪無8**）・アイテム20種・スペル25種。各カードに★〜★★★★の**レア度**（アルバム/カードに星表示）
- **⚪ 無属性クリーチャー**（全員レア以上）: 属性相性の輪の外＝有利も不利も取らず**土地の加護も受けない**。
  そのぶんコスト効率が高くユニークな能力を持つ——**ガーゴイル**（守護＋護法）・**ユニコーン**（先制＋豪運）・
  **ミスリルゴーレム**（護法）・**キメラ**（連撃＝2回攻撃）・**ファントム**（物理無効・HP低め）・**ミラージュ**（物理反射・HP極小）・
  **🎭ドッペルゲンガー**（模倣＝相手と同じ強さで戦うレジェンド）・**スフィンクス**（魔法攻撃＋守護の番人）
- **✨ 物理と魔法**: ファントム/ミラージュには通常の（物理）攻撃が通らない。対策は**魔法攻撃**——
  各属性の術士（フレイムメイジ/ドルイド/ルーンドワーフ/フロストウィザード＝いずれもHP低め）か、
  **マジックワンド／アルカナロッド**（武器よりST補正は低いが攻撃が魔法になるアイテム）、または☄️メテオ等の除去スペル
- 逆転系スペル: **リベンジ**・**リコール**・**引き直し**・**スネアトラップ**に加え、
  **💰プランダー**（相手の所持金の半分を奪う）・**🎲ダイスブースト**（次の出目を2倍・最大12）
- **移動スペル（v17）**: **💫テレポート**（自分のコマを好きなマスへ。城以外・その後ダイスで移動）・
  **🚪トランスポート**（自分のクリーチャーを好きな空き地へ転送）・**🐇リープ**（自分のクリーチャーを2マス先の空き地へ跳躍）
- 新アイテム: **🚫ディスペルワード**（相手のアイテムを打ち消す）・**🪞ミラーシールド**（受けた攻撃の50%を反射）・
  **✨マジックワンド／アルカナロッド**（攻撃が魔法になる）・**💰グリードファング**（ST+25・与えたダメージ×2倍の魔力を強奪・v17）
- スペルはダイスを振る前に手札のカードをクリックで使用（1ターン1回）
- **📚コレクション**: ステージ初クリアで10枚・正規勝利で5枚・🎯トレーニング勝利で3枚のカード獲得。
  トレーニングは🔥**3連勝から毎回レア以上1枚保証**（負け・投了でリセット）。
  🛠自分だけの30枚デッキを構築できる（🎯トレーニングは時短で気軽に周回）
- **👤プレイヤー選択**: 最大5人が**別々のコレクション・デッキ・ステージ進行度**で遊べる
  （名前変更可）。デッキはプレイヤーごとに**5つまで保存**でき、使うデッキを切り替えられる
- **♻️カード工房**: 同名4枚目以降の余剰カードを**🔮マナの欠片**に分解し、欠片で**好きなカードを生成**
  （分解 ★+1〜★★★★+8／生成 ★4〜★★★★32）。未所持カードも作れるのでアルバムコンプの出口になる

### 見た目・演出（v14〜v16）
- **タイトル画面**: マナの粒子が瞬く夜空にゆっくり回る大紋章、地平にはクリーチャーたちのシルエット。
  クリック／タップで冒険が始まる
- **ステージ選択**: 紋章と口上を掲げた「遠征の書」。トレーニングは「修練の間」、2人対戦は「決闘の間」
- **カード**: 全カードにアート（クリーチャーは名前に合った影絵の造形＋魔力の光沢）。
  裏面は**守護竜とマナ回路の紋章**——クリーチャーアートと同じ影絵様式で統一されたカードバック。
  めくり演出のカードは表裏とも同寸（TCG比率の固定サイズ）
- **開幕**: 手札が表紙側で配られ、1枚ずつめくれて対戦スタート。毎ターンのドローもめくって手札へ
- **バトル**: 両者が左右から突撃する**カットイン演出**（突進・被弾・会心のフラッシュ）。⏩でスキップ可
- **パック開封**: 封のされたパックをクリックで開封 → カードを1枚ずつめくる（初入手はNEW・レア以上は光る）
- **勝利**: 金の光条と紙吹雪が舞う祝福演出
- **盤面**: マスをつなぐ「回路の道」、属性ごとのグラデーションタイル、宝珠風の駒（手番側は光が脈動）

### モード・その他
- **🎮 2人対戦（ホットシート）**: 同じ端末を交互に操作する人間同士の対戦。プレイヤーを2人選び、
  **全ステージ**から盤面を選べる。各自の使用中デッキで戦い、手番交代時は手札が伏せられる（報酬・進行度は変化なし）
- **🎪 ウィークリールール**: 毎週月曜に切り替わる特殊ルール（通行料2倍・初期手札レジェンド保証・出目4〜6のみ 等の全7種）。
  タイトルの「🎪 週替り」でON/OFF。ONで正規対戦に勝つと**ボーナスカード+2枚**（トレーニングには適用されない）
- **🎵 BGM**: ヘッダーの🎵でON/OFF。効果音と同じくWebAudioで自動生成（オフライン動作・外部ファイル不要）
- **📱 スマホ最適化（v17）**: 手札は**◀▶ボタンで1枚ずつ送れる**（横スワイプ不要＝Androidの「戻る」ジェスチャーと衝突してゲームが終了する事故を防ぐ。手札カードも縮小表示）。
  ヘッダーの**「⛶ 全体」ボタン**で盤面全体が画面に収まる倍率へワンタップ調整（対戦開始時も自動フィット）
